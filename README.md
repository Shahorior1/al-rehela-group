# Al Rehela Group Website

A professional construction company website showcasing Al Rehela Group's services, projects, and expertise in trading, contracting, and construction services.

## Features

- **Responsive Design**: Fully responsive layout that works on desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean, professional design with burgundy color scheme
- **Service Showcase**: Comprehensive display of construction services including:
  - Manpower Supply
  - Gypsum Installation
  - Interior Design
  - Electrical Works
  - Civil Works
  - Aluminum Works
  - Painting Works
  - Materials Supply
  - Tiles and Marble Fixing
  - Steel Works
  - Plastering
  - Cleaning

- **Project Gallery**: Visual showcase of completed projects
- **Team Section**: Meet the team members and their roles
- **Contact Form**: Interactive contact form for customer inquiries
- **Social Media Integration**: Links to social media platforms

## Technologies Used

- **HTML5**: Semantic markup structure
- **CSS3**: Modern styling with flexbox and grid layouts
- **JavaScript**: Interactive functionality and form handling
- **Font Awesome**: Icons for social media and UI elements
- **SVG Graphics**: Scalable placeholder images

## Getting Started

1. Open `index.html` in a web browser
2. The website is fully functional with:
   - Responsive navigation
   - Interactive contact form
   - Smooth scrolling
   - Mobile-friendly menu

## Customization

### Colors
The main brand colors are defined in CSS:
- Primary: `#8B1538` (burgundy)
- Secondary: `#A91B47` (lighter burgundy)
- Accent: `#007bff` (blue for buttons)

### Content
- Update company information in `index.html`
- Modify contact details in the footer section
- Add real images by replacing the SVG placeholders

### Contact Information
Update the contact information in the footer:
- Phone number
- Email address
- Physical address
